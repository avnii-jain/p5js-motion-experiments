# 🎨 p5.js Motion Experiments

This repository contains a set of beginner-friendly experiments built using p5.js to explore motion, animation, and mathematical visualization.

## 🚀 Overview

The goal of this project is to understand how simple mathematical concepts like sine and cosine can be used to create dynamic visual patterns. These sketches demonstrate how animation works in p5.js using the `draw()` loop and `frameCount`.

---

## 📂 Projects

### 1. Basic Circular Motion
A single circle moves in a circular path using trigonometric functions.

**Concepts used:**
- `sin()` and `cos()`
- `frameCount` for animation
- Coordinate system

---

### 2. Multiple Circular Motion
Multiple circles move along circular paths with different radii, creating layered motion patterns.

**Concepts used:**
- Loops (`for`)
- Dynamic radius variation
- Pattern formation using math
